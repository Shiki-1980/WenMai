# 篇章概述
失忆青年[[洛辰·星语]]在[[翡翠森林]]边缘的[[晨辉镇]]苏醒，在寻找身份的过程中成为冒险者，与游侠[[叶琳娜·风行者]]组成小队探索[[迷雾沼泽]]遗迹。他们发现[[混乱之井]]碎片，遭到[[暗影议会]]势力追杀，在逃亡中结识[[矮人工匠同盟]]大师[[巴利姆·铁砧]]，得知世界正面临[[混沌守恒]]崩溃的危机。洛辰决定前往[[星辰港]]寻求[[自由冒险者公会]]的帮助，正式踏上阻止世界毁灭的征程。

# 约束条件详细说明

1. **主角实力限制**：[[洛辰·星语]]在篇章全程保持[[源能学徒]]阶位，直到结尾才升至[[源能行者]]下品。[[源能共鸣]]能力在本篇章中仅能激活3次，每次使用后必须间隔7天且伴随反噬（轻微：局部植物枯萎、天气忽变；中等：短暂昏厥、内伤；重度：大片土地荒芜）。反噬效果不可消除，仅能通过休养缓解。

2. **记忆恢复进度**：洛辰的记忆只能碎片式恢复，本章中仅能回忆起自己是[[光之女神]]守护者以及因拒绝参与净化行动被[[审判所]]放逐。其他身份细节（如[[神子级]]具体实力、过往人际关系）严禁在本篇章中全部揭晓，需留待后续篇章。

3. **反派战力梯度**：本章反派人物的单体战力不得超过[[源能行者]]上品。[[阴影刺客莫里斯·暗刃]]为[[源能行者·上品]]，[[恶魔召唤师阿萨斯·血火]]为[[源能导师·下品]]，[[魔尊·阿兹莫德]]的分身投影仅为[[源能导师·上品]]水平。所有高过此线的角色（如[[尤利娅·风暴之子]]、[[魔尊真身]]）只能作为出场展示或背景提及，不得参与战斗。

4. **地理范围限制**：本章节剧情活动范围严格限制在[[翡翠森林]]东部、[[迷雾沼泽]]北部、[[星辰港]]市区及其通往[[死寂沙漠]]的北大道。其他区域（如[[圣光城]]、[[龙脊山脉]]、[[暗影深渊]]第5层以下）只能通过情报、对话、闪回提及，主角队伍不得实际进入。

5. **重要角色存活保障**：[[叶琳娜·风行者]]、[[巴利姆·铁砧]]、[[尤利娅·风暴之子]]在本篇章中必须存活，不得遭受导致永久性残疾或实力永久降级的伤害。[[洛辰·星语]]不得在本篇章中使用[[命运匕首]]或展现[[源能宗师]]级能力。

6. **伏笔揭晓节奏**：每10章必须至少揭晓1个此前埋下的伏笔（例如第12章揭晓[[锈蚀徽章]]的部分秘密），同时埋设至少1个新伏笔。本章结尾时，[[审判所所长洛伦佐]]背叛和[[源能锚点]]抹除存在的两大主线伏笔必须已被引入。
